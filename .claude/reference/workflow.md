# 📘 Claude Code Development Workflow

## 1. Objective

Define: - Project name - High-level goal - Expected deliverables -
Timeline

------------------------------------------------------------------------

## 2. Research Phase

Before writing code: - Ask Claude to deeply read relevant
folders/files - Generate a research.md document - Document: - How the
system works - Dependencies - Edge cases - Potential bugs

------------------------------------------------------------------------

## 3. Planning Phase

Create a detailed plan.md including: - Step-by-step implementation
plan - Exact files to modify - Code snippets - Impact analysis

Important: Do NOT implement yet.

------------------------------------------------------------------------

## 4. Annotation Cycle

Refine the plan before coding: - Add comments - Clarify uncertainties -
Adjust architecture

Repeat until fully confident.

------------------------------------------------------------------------

## 5. Todo List

Add granular tasks to plan.md: - Task 1 - Task 2 - Task 3

This becomes your progress tracker.

------------------------------------------------------------------------

## 6. Implementation Phase

Only now allow Claude to implement: - Follow plan.md strictly - Mark
completed tasks - Run type checks and tests continuously

------------------------------------------------------------------------

## 7. Feedback Loop

Cycle: Claude implements → You review → Claude corrects

Keep feedback precise and short.

------------------------------------------------------------------------

## 8. Long Session Strategy

Keep research → plan → implement in one continuous session to preserve
context.

------------------------------------------------------------------------

## 9. Core Principle

Never let Claude write code before a reviewed written plan exists.

------------------------------------------------------------------------

## 10. Benefits

-   Reduced wasted work
-   Better architectural clarity
-   Persistent project memory
-   Higher quality outputs
